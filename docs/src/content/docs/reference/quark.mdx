## 签到 URL 获取

- 通过手机抓包签到过程
- 活动界面找到类似，注意后面 query 需要完整 `https://drive-m.quark.cn/1/clouddrive/capacity/growth/info?__t=1720880459186&sign_cyclic=true&kps=xxxxxxxxxxxxxxx`，从链接中提取出 `vcode`、`kps` 和 `sign`
- 当然也可以直接查看 headers 中的 `x-u-kps-wg`（对应 kps）、`x-u-sign-wg`（对应 sign）和 `x-u-vcode`（对应 vcode）
